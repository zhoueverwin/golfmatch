# GolfMatch - ゴルフマッチ

## アプリ概要

GolfMatch（ゴルフマッチ）は、ゴルフを愛する人々のためのマッチングアプリです。ゴルフという共通の趣味を通じて、新しい出会いやパートナーを見つけることができます。日本全国のゴルファーが集まり、ゴルフラウンドのパートナー探しから、より深い関係性の構築まで、様々な出会いの可能性を提供します。

## 主な機能

### 1. ユーザープロフィール

- **基本情報**: 名前、年齢、性別、居住地（都道府県）、血液型、身長、体型、喫煙習慣など
- **ゴルフ情報**: 
  - スキルレベル（ビギナー、中級者、上級者、プロ）
  - ゴルフ経験年数
  - 平均スコア
  - 交通手段
  - プレー料金の希望
  - プレー可能日（カレンダー機能）
- **プロフィール写真**: 複数枚の写真をアップロード可能
- **自己紹介文**: 自由に自己紹介を記入
- **本人確認バッジ**: KYC認証により、本人確認済みバッジを表示

### 2. マッチング機能

- **いいね**: 気になるユーザーに「いいね」を送信
- **スーパーいいね**: 特別な興味を示す「スーパーいいね」機能
- **パス**: 興味がない場合はスキップ
- **マッチング**: お互いに「いいね」を送った場合、マッチング成立
- **マッチング通知**: マッチング成立時に通知とお祝いアニメーションを表示

### 3. メッセージ機能

- **リアルタイムチャット**: マッチングしたユーザーとメッセージのやり取りが可能
- **画像・動画の送信**: チャット内で写真や動画を共有
- **メッセージ通知**: 新しいメッセージを受信した際にプッシュ通知
- **メッセージ履歴**: 過去の会話を確認可能

### 4. ソーシャルフィード（ホーム）

- **投稿機能**: ゴルフに関する投稿を共有
- **画像・動画投稿**: 写真や動画を添付して投稿
- **いいね機能**: 投稿に「いいね」を付ける
- **ユーザー投稿一覧**: 特定のユーザーの投稿を一覧表示

### 5. ユーザー検索・発見

- **おすすめユーザー**: アルゴリズムに基づいたおすすめユーザーを表示
- **新規登録順**: 新しく登録したユーザーを確認
- **詳細フィルター**:
  - **居住地**: 47都道府県から選択
  - **年齢**: 20代、30代、40代、50代、60代、70代以上から複数選択可能
  - **ゴルフスキルレベル**: ビギナー、中級者、上級者、プロ
  - **平均スコア**: 80以下、90以下、100以下、110以下、120以下、130以下、指定しない
  - **最終ログイン**: 24時間以内、3日以内、7日以内、30日以内、指定しない
- **フィルター保存**: 設定したフィルターを保存して次回も使用可能

### 6. つながり管理

- **いいね一覧**: 自分が送った「いいね」と「スーパーいいね」を確認
- **マッチング一覧**: マッチングしたユーザーを一覧表示
- **過去のいいね**: 過去に送った「いいね」の履歴を確認
- **足あと機能**: 自分のプロフィールを閲覧したユーザーを確認（実装予定）

### 7. カレンダー機能

- **プレー可能日の設定**: カレンダー上でゴルフプレー可能な日を設定
- **空き状況の共有**: 他のユーザーに自分の空き状況を表示

### 8. 本人確認（KYC認証）

- **3ステップ認証**:
  1. 身分証の写し（運転免許証、パスポートなど）
  2. セルフィー（自撮り写真）
  3. 身分証とセルフィーの組み合わせ写真
- **画像品質チェック**: アップロード前に画像の品質を自動検証
- **管理者による審査**: 提出された写真を管理者が手動で審査
- **認証バッジ**: 認証完了後、プロフィールに「本人確認済み」バッジを表示
- **再提出機能**: 審査で不承認になった場合、再提出が可能

### 9. 通知機能

- **プッシュ通知**: 
  - 新しいマッチング
  - 新しいメッセージ
  - いいね通知
  - 投稿へのいいね
  - その他のアクティビティ
- **通知設定**: 各種通知のオン/オフを個別に設定可能
- **通知履歴**: 過去の通知を一覧表示

### 10. 設定・アカウント管理

- **プロフィール編集**: いつでもプロフィール情報を更新
- **アカウント連携**: 
  - Googleアカウント連携
  - Apple ID連携
  - メールアドレス連携
- **通知設定**: 詳細な通知設定
- **ヘルプ・サポート**: よくある質問やお問い合わせ機能
- **利用規約・プライバシーポリシー**: アプリの利用規約を確認

## 認証方法

GolfMatchでは、以下の認証方法に対応しています：

1. **電話番号認証（OTP）**: SMSで送信される認証コードを使用
2. **メールアドレス・パスワード**: 従来のメールアドレスとパスワードでのログイン
3. **Googleアカウント**: Googleアカウントで簡単ログイン
4. **Apple ID**: Appleアカウントで簡単ログイン（iOS）

## メンバーシッププラン

GolfMatchでは、メッセージ機能を使用するためにメンバーシッププランへの加入が必要です（女性ユーザーは無料で利用可能）。

### ベーシックプラン

- **価格**: ¥3,000（月額・税込）
- **機能**: 
  - メッセージの送受信が可能
  - マッチングしたユーザーとチャット可能
- **支払い方法**: iTunesアカウント（iOS）またはGoogle Playアカウント（Android）
- **自動更新**: 月次で自動更新（設定でキャンセル可能）

### 永久プラン

- **価格**: ¥10,000（一回限り・税込）
- **機能**: 
  - メッセージの送受信が永続的に可能
  - 月額料金不要で永久に利用可能
- **支払い方法**: iTunesアカウント（iOS）またはGoogle Playアカウント（Android）

### 女性ユーザー向け特典

女性ユーザーは、メンバーシッププランに加入しなくても、すべての機能を無料で利用できます。これは、プラットフォームのバランスを保つための施策です。

### 購入の復元

アプリを再インストールした場合や、新しいデバイスに移行した場合でも、「購入を復元」機能を使用して、以前の購入を復元できます。

## 技術仕様

### プラットフォーム

- **iOS**: iPhone、iPad対応
- **Android**: Androidスマートフォン、タブレット対応

### 使用技術

- **フレームワーク**: React Native (Expo SDK 54)
- **バックエンド**: Supabase（認証、データベース、リアルタイム通信、ストレージ）
- **状態管理**: React Context + TanStack Query (React Query v5)
- **ナビゲーション**: React Navigation（Stack + Bottom Tabs）

### データベース

- **ユーザープロフィール**: 基本情報、ゴルフ情報、写真など
- **マッチングデータ**: いいね、マッチング、メッセージ履歴
- **投稿データ**: ソーシャルフィードの投稿とリアクション
- **通知データ**: プッシュ通知の管理
- **メンバーシップデータ**: 購入履歴と有効期限管理

## プライバシーとセキュリティ

- **データ暗号化**: すべての通信は暗号化されています
- **本人確認**: KYC認証により、なりすましを防止
- **プライバシー設定**: プロフィールの公開範囲を設定可能
- **データ保護**: 個人情報は適切に管理され、第三者に提供されることはありません

## サポート

アプリ内の「ヘルプ」セクションから、よくある質問を確認したり、お問い合わせを送信することができます。お問い合わせには、通常24-48時間以内に返信いたします。

## 利用規約

アプリを利用する前に、利用規約とプライバシーポリシーを必ずお読みください。アプリ内の「設定」→「プライバシーポリシーと利用規約」から確認できます。
